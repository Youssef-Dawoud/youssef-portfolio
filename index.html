02 — Research

Master's Thesis: AgnosticSSL

Extending a backend-agnostic OpenSSL engine

OpenSSL Engine OP-TEE OpenTitan / RISC-V GlobalPlatform TEE Client API ECDSA Ephemeral ECDH ChaCha20 RNG

AgnosticSSL is a modular OpenSSL engine that transparently redirects cryptographic operations into heterogeneous trusted environments — currently OP-TEE and OpenTitan, a RISC-V silicon Root of Trust — so applications get hardware-backed isolation with zero code changes. On OpenTitan, cryptographic offload has shown speedups of up to 40× for SHA-256 and 82× for AES-256-CBC over software execution.

I'm continuing this work, building on the foundation laid by T. Sayarmoafi (Politecnico di Torino, 2025) under Prof. L. Barbierato. My contribution broadens the engine's cryptographic coverage toward full TLS 1.3 support: elliptic-curve signatures (ECDSA), ephemeral key exchange (ECDH), and a ChaCha20-based random number generator for secure ephemeral key generation.

Supervisor: Prof. L. Barbierato — Politecnico di Torino
